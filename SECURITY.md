# Security Policy

## Supported Versions
| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
Please report security vulnerabilities by emailing security@codieverseplus.com. Do not open public issues for security vulnerabilities. We will review your report within 48 hours and provide a timeline for remediation.
